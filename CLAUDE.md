# Skills

Setiap pesan atau perintah masuk → otomatis cocokkan dengan skill di bawah → jalankan skill yang sesuai tanpa perlu diminta. Kalau tidak ada yang cocok, kerjakan langsung.

| Skill | Gunakan ketika |
|-------|----------------|
| `web-design-guidelines` | review UI, audit tampilan, cek aksesibilitas, "sudah bagus belum?" |
| `frontend-design` | bangun atau ubah desain visual, arah estetik, tipografi |
| `performance-optimization` | performa lambat, LCP/CLS buruk, ada bottleneck |
| `spec-driven-development` | fitur baru atau perubahan besar yang belum jelas requirement-nya |
| `ponytail` | "buat simpel", "jangan lebay", "cara paling pendek", solusi minimal |
| `find-skills` | "ada skill untuk X?", cari kemampuan yang belum ada |
